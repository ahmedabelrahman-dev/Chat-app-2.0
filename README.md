## Overview

This is a real-time chat application built using MERN stack (MongoDB, Express.js, React, Node.js). It allows users to create an account, login, and chat with other users in real-time.

## Features

- User authentication and authorization
- Real-time messaging
- User profiles
- User search and conversation history

## Usage

1. Open the application in your browser
2. Create an account or login
3. Start chatting with other users
